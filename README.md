# Дипломный проект:  SPA - приложение "Movies-explorer"

#Функционал
- авторизации и регистрации пользователей
- поиск фильмов по ключевым словам
- фильтр фильмов по продолжительности
- добавление/удаление фильмов в избранное

#Стэк
- React
- Express.js
- MongoDB
